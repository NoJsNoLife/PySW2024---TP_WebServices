Resumidor de textos
  1. Utilizando la API de AI Summarize, podemos resumir textos de cualquier tipo.
  2. Ingrese en el input una URL de algún texto a su gusto que desee resumir.
  3. Presione el botón "Resumir" para enviar la petición a la API.
  4. Se mostrará el resumen en el cuadro de 'textarea'.

    5. **Está limitado a 5 resumenes por día**
